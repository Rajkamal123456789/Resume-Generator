# Professional Resume Builder

A dynamic and interactive web-based Resume Builder application that allows users to create professional resumes with ease. The application features a real-time preview and PDF export functionality.

## Features

- **Real-time Preview**: See your resume update instantly as you type.
- **Dynamic Sections**: Add as many entries as needed for:
  - Education
  - Work Experience
  - Projects
  - Achievements & Leadership
- **Customizable Skills**: Select from a predefined list or add custom technical skills.
- **PDF Export**: Download your finalized resume as a high-quality PDF document.
- **Single Page Layout**: Optimized to fit content seamlessly into a continuous layout.
- **Responsive Design**: Works on desktop and mobile devices.

## How to Use

1. **Open the Application**: Open `index.html` in any modern web browser.
2. **Fill in Details**:
   - Enter your Personal Details (Name, Email, Phone, Links).
   - Write a Career Objective.
   - Add your Education history.
   - Select and add Technical Skills.
   - Add Work Experience, Projects, and Achievements using the "+ Add" buttons.
3. **Review**: Check the live preview on the right side (or bottom on mobile) to ensure everything looks correct.
4. **Download**: Click the "Download PDF" button to save your resume.

## Technologies Used

- **HTML5**: For structure and semantic markup.
- **CSS3**: For styling, layout (Grid/Flexbox), and print optimization.
- **JavaScript (Vanilla)**: For dynamic DOM manipulation and logic.
- **html2pdf.js**: For generating PDF documents from HTML content.

## Project Structure

- `index.html`: Main application structure.
- `style.css`: Styling for the form and resume preview.
- `script.js`: Logic for handling form inputs, dynamic updates, and PDF generation.

## Setup

No installation required. Simply clone the repository or download the files and open `index.html` in your browser.
